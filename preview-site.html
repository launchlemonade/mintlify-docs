Lemonade Types

Choose the right type of Lemonade for your use case

Overview

There are five types of Lemonades you can create - each does something a bit different. Most people start with Chat Lemonades (the most popular and easiest), but feel free to explore!

The Five Types

1. Chat Lemonade

The most popular choice. Build AI assistants that chat with users.

Best for: Customer support bots, Q&A assistants, Personal AI helpers, Internal knowledge bases

Features: Access to 20+ AI models, Upload your own knowledge, Customize personality, Deploy via embed, email, or API

New to LaunchLemonade? Start here! Chat Lemonades are the easiest to set up and most versatile.

2. Voice Lemonade

AI that listens and speaks. Natural voice conversations.

Best for: Voice-based coaching, Call center automation, Accessibility applications, Interactive voice experiences

Features: Speech-to-text input, Text-to-speech output, Empathetic voice models, Natural conversation flow

3. Lemonade Flows

Multi-step AI workflows with form-like experiences.

Best for: Lead capture forms, Customer surveys, Onboarding processes, Data collection

Features: Step-by-step conversation paths, Conditional logic (if/then branching), Data collection and export

4. Lemonade Agents

Combine multiple Lemonades for complex, autonomous tasks.

Best for: Complex automation, Multi-tool workflows, Agentic AI systems

Agents are powerful but have more moving parts. Once you're comfortable with Chat Lemonades, Agents will feel much easier to tackle.

5. Image Lemonade

Generate and edit images with AI.

Best for: Visual content creation, Marketing graphics, Product mockups, Creative exploration

Knowledge & Training

Train your Lemonade with files, websites, and integrations

Why Knowledge Matters

Want your Lemonade to know YOUR business? The Knowledge tab is where you share your documents, websites, and data - so your AI can give answers specific to you, not generic internet responses.

Good knowledge makes your Lemonade smarter, more accurate, and genuinely helpful for your unique needs.

üñºÔ∏è Screenshot: Knowledge Tab

Three Ways to Add Knowledge

1. Add Files

Upload documents directly to train your Lemonade.

Click Add Files
Select Documents
Wait for Processing

Supported File Types:

FormatExtensionBest For
PDF.pdfManuals, reports, ebooks
Word.docxDocuments, SOPs
Text.txtSimple text content
CSV.csvData tables, lists
Excel.xlsxSpreadsheets
Break large documents into smaller, focused files for better results. A 100-page manual should be split into logical sections.

2. Add Websites

Crawl and learn from web content.

Great for: Product documentation, FAQ pages, Help center articles, Blog content, Company information

Only crawl websites you own or have permission to use. When in doubt, stick to your own content!

3. Integrations

IntegrationDescriptionUse Case

Google DriveConnect your Drive filesShared documents, spreadsheets

Knowledge vs Brain

FeatureKnowledge (Tab 4)Brain (Profile & Billing)
ScopeThis Lemonade onlyAll your Lemonades
ContentDocuments, websitesPersistent memory
Use CaseDomain expertiseRemembered facts
UpdatesManual uploadsAutomatic learning
Use Knowledge for specific expertise (product docs, FAQs). Use Brain for things that should be remembered across all conversations.

For Best Results

Start small - A few focused documents work better than uploading everything at once.

Keep it fresh - Update files when information changes.

Remove duplicates - Don't upload the same content multiple ways.

Use clear names - Name files descriptively.

Quality over quantity - 10 well-organized pages beat 100 poorly structured ones.

Troubleshooting

File Won't Upload

  • Check file size (max 25MB)
  • Verify file format is supported
  • Try a different browser

Website Won't Crawl

  • Verify the URL is accessible
  • Check if the site allows crawling
  • Try a specific page instead of the homepage

Overview

Want to connect your Lemonade to your own app or automate it with tools like Zapier? Developer Mode gives you API access to do exactly that.

Who This Is For

This page is for developers who want to integrate Lemonades into applications, workflows, or automation systems.

Not a developer? That's OK! You can still use Zapier integration without writing code. Skip to the Zapier Integration section.

Enabling Developer Mode

Open Lemonade Editor
Go to Add On Tab
Enable Developer Mode

Toggle Developer Mode to On

Getting Your API Key

Go to Profile & Billing
Navigate to API Keys
Generate Key

Click "Generate New Key"

Important: Your API key is like a password - keep it secret! Don't share it publicly or save it in code repositories.

API Endpoints

Send Message

POST https://api.launchlemonade.app/v1/chat

Request Body

{
  "lemonade_id": "your-lemonade-id",
  "message": "Hello, how can you help me?",
  "conversation_id": "optional-conversation-id"
}

Response

{
  "response": "Hello! I'm here to help you with...",
  "conversation_id": "conv_123abc",
  "tokens_used": 150
}

Code Examples

Python

import requests

response = requests.post(
    "https://api.launchlemonade.app/v1/chat",
    headers={"Authorization": f"Bearer {API_KEY}"},
    json={"lemonade_id": LEMONADE_ID, "message": "Hello!"}
)
print(response.json()["response"])

JavaScript

const response = await fetch('https://api.launchlemonade.app/v1/chat', {
  method: 'POST',
  headers: {'Authorization': `Bearer ${API_KEY}`},
  body: JSON.stringify({lemonade_id: 'your-id', message: 'Hello!'})
});
const data = await response.json();
